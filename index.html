About

Environmental scientist and geospatial data scientist with a career built around one core instinct: take messy, large-scale data and extract findings that actually mean something. PhD research at UQTR (2021–2026) focuses on non-binary monitoring of seasonally frozen ground using passive microwave remote sensing — developing probabilistic frameworks that replace decades of oversimplified binary assumptions in cryosphere science.

Every project follows the same logic: get the raw data (however dirty, however large, whatever format), build a rigorous pipeline that handles the edge cases and the RAM crashes, apply the right statistical or machine learning tool for the job, and turn the result into something a scientist or a general audience can actually use.

Comfortable across the full technical stack: production-scale geospatial pipelines (xarray, dask, Zarr, Parquet), statistical and probabilistic modelling (PyMC, XGBoost, scikit-learn), remote sensing data engineering (GDAL, rasterio, HDF5/NetCDF), and rigorous model evaluation. Currently at Agriculture and Agri-Food Canada as a Geospatial Data Scientist, and building toward cloud deployment of geospatial ML pipelines.
